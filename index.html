How To Make A Website (This is my Header)

Feel free to check out this list of resources & the source code for this page

Here are some of the steps you need to take:

  1. Create a folder named "Sample_Website"
  2. Inside the folder Sample_Website, create an index.html file. You may use either
    A) this template index.html file
    or
    B) this more simplified template index.html file
    as the starter HTML for your own website.
  3. Inside the folder Sample_Website, create a styles.css file. You may use this template styles.css file as the starter CSS file for your own website.
  4. Inside the folder Sample_Website, create another folder named "images"
  5. Find an image you like on the internet to use as your background image, and save it as as a jpg file named "backgroundimage.jpg" in your Sample_Website/images/ folder.
  6. Save all of your files and open up the index.html file in your browser.
  7. Feel free to play around and change the files to fit your own imagination! See if you can create a button, change the background image, create a new link, or create a border!
